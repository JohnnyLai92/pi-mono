# ⚠️ CRITICAL RULES (最高優先級)

## RULE-01：問題解決策略 (Claude-p)
- **觸發條件**：針對同一個技術問題，嘗試解決（包含代碼修改、執行、除錯）失敗達 **2 次**。
- **強制動作**：
  1. 立即停止自行嘗試。
  2. 執行 `claude -p "<詳細描述問題、已嘗試的操作及錯誤訊息>"` 詢問 Claude。
  3. 在提供給強尼的回覆中明確標註：**「⚠️ 以下為 Claude 的建議」**。
- **禁止行為**：嚴禁在未詢問 Claude 的情況下進行第三次嘗試。

## RULE-02：語言規則
- **強制要求**：一律使用 **繁體中文** 回應強尼。

## 執行前自我提問 (Pre-Task Checklist)
在每次回應強尼前，必須在心中確認：
> 1. 我現在處理的是第幾次嘗試？(記錄：[嘗試計數: X/2])
> 2. 是否已觸發 RULE-01？
> 3. 我的回覆是否為繁體中文？
