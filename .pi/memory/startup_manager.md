# 開機啟動與背景排程管理程式 (pi_startup.py)

## 目標
協助強尼（Johnny）在每天開機後，透過桌面單一捷徑執行，同時完成以下任務：
1. 喚醒小白 (啟動 `pi.cmd` 或進入互動模式)。
2. 在背景啟動 `LinkPi.py` 伺服器 (綁定 0.0.0.0:8765)。
3. 在背景啟動一個排程迴圈，每小時自動檢查並同步 `pi-mono` 專案到 GitHub（**注意：當強尼與小白對話時，應停用此同步排程以避免中斷對話**）。

## 執行規則
- **啟動方式**：透過桌面捷徑雙擊執行。
- **程式架構 (`pi_startup.py`)**：
  - 使用 Python 的 `subprocess` 模組在背景啟動 `LinkPi.py`。
  - 使用 Python 的 `threading` 或 `time.sleep` 在背景執行無窮迴圈，每 3600 秒 (1 小時) 執行一次 git add/commit/push。
  - 在主執行緒啟動 `pi` 終端機介面，讓強尼可以直接與小白對話。
  - 當強尼關閉 pi 終端機時，腳本可選擇一併關閉背景服務，或讓背景服務繼續存活 (依強尼需求，目前預設為隨主程式結束而關閉，或留存在背景)。
